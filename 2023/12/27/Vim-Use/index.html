Vim 使用记录

记录在使用vim的过程中遇到的遇到的问题和解决办法，或者想要完成的操作

一 使用技巧

1 批量替换

推荐阅读：Find and Replace in Vim/Vi

  • 基础操作

替换命令的格式为:[range]s/{pattern}/{string}/[flags] [count]

表示将[range]s中的每一行符合{pattern}的字符串替换为{string}，并重复该操作[count]次，在未指定[range]和[count]是默认在当前行执行一次，例如有如下操作

:s/foo/bar将当前行的foo替换为bar

  • 常用操作

:%s/foo/bar/gci将整个文件中的foo替换为bar，其中%表示在整个文件中搜索，g表示处理每一个匹配到的字符串，c表示需要确认，i表示大小写不敏感

:.,+4s/foo/bar/g将当前行后面4行的foo替换为bar，.表示当前行，+4表示后面四行

:s/\<foo\>/bar/表示将foo这个单词替换为bar，<foo>表示搜索单词

:s可以进入substitute的历史记录

:%s/\s+$//e表示删除每行的末尾空格，e表示没有搜索到也不报错，\s在正则表达式中表示空格

2 多行操作（Visual mode）

推荐阅读：Mastering Visual Mode in Vim

Visual mode can be useful if you want to target a specific group of text that doesn't follow distinguishable pattern.

  • 进入visual mode

Vim的visual mode有三种模式，通过输入：v进入字符模式，V进入行模式，Ctrl-v进入块模式

在块模式下，vim允许使用者按照行列来选取一个块

gv可以进入上一次进入的visual模式，并保持上一次选取的内容

o在visual mode中移动光标到选中部分的开始或结尾，而不改变选中部分

  • visual mode下的操作

在选择需要操作的部分后，可以执行如下操作：u变为小写，U变为大写，d删除，c改变，y复制，>缩进，<回缩，ctrl-a/ctrl-x用于增加或者减少光标选中的数字

ex command（例如subsitute命令）在visual模式下，会默认在选中部分执行

.表示重复上一次操作

在块模式下，选中要操作的多个行后，键入A或者I进行输入，然后推出visual mode，该输入操作会在选中的每个行复制

在insert mode下，可以通过ctrl-o + command执行normal mode下的命令，例如ctrl-o v可以进入visual mode

  • select mode

可以通过gh gH gCtrl-h进入和visual mode相同的几个select mode，在select mode下，和正常的文本编辑器类似，可以直接输入某个字符替换选中的部分
